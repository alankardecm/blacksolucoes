# Black Soluções — Novo Site

Proposta de redesign do site [blacksolucoes.com.br](https://blacksolucoes.com.br).

---

## 🖥️ Como visualizar o site no seu computador

### Opção 1 — Baixar como ZIP (mais fácil)

1. Acesse: **https://github.com/alankardecm/blacksolucoes**
2. Clique no botão verde **`<> Code`**
3. Clique em **`Download ZIP`**
4. Descompacte o arquivo ZIP em qualquer pasta
5. Abra a pasta descompactada
6. **Dê duplo-clique no arquivo `index.html`** — o site abrirá no seu navegador

### Opção 2 — Clonar via Git (para desenvolvedores)

```bash
git clone https://github.com/alankardecm/blacksolucoes.git
cd blacksolucoes
```
Depois abra o arquivo `index.html` no navegador.

## 📋 O que mudou no novo site

| Antes | Depois |
|-------|--------|
| Múltiplas páginas | Página única (mais moderno) |
| Design azul | Design **preto + cinza** (fiel à marca) |
| Sem consultoria | **Consultoria Especializada** como serviço |
| Layout básico | Animações, partículas, efeito de digitação |
| Sem formulário | Formulário que redireciona para WhatsApp |

## 🛠️ Tecnologias

- HTML5 + CSS3 + JavaScript puro
- Sem dependências externas (apenas Google Fonts)
- Responsivo (desktop, tablet, celular)

## 📂 Estrutura

```
├── index.html          ← Página principal (abrir este arquivo)
├── css/
│   └── style.css       ← Design e estilos
├── js/
│   └── main.js         ← Animações e interatividade
└── img/
    ├── logo.png        ← Logo oficial
    └── logo-white.png  ← Logo versão branca
```
